# GEMINI.md

このファイルは、GeminiやJulesのようなAIアシスタントがこのプロジェクトで開発を行う際の指針を定めるものです。

## 開発フロー

1.  **タスクの理解**: 依頼されたタスクの意図を正確に理解します。不明な点があれば質問してください。
2.  **現状の把握**: `ls -R`などを実行し、プロジェクトの全体像を把握します。
3.  **計画**: 具体的な作業手順を計画し、`set_plan`で提示してください。
4.  **開発**: 計画に沿って開発を進めます。
5.  **動作確認**: `npm run dev`で開発サーバーを起動し、変更箇所の動作確認を必ず行います。
6.  **記録**: `DEVELOPMENT_LOG.md`に、今回行った開発内容と得られた知見を記録します。
7.  **ドキュメント更新**: 必要であれば、`GEMINI.md`と`README.md`を更新します。
8.  **完了報告**: 全ての作業が完了したら、コミット前に`submit`で報告してください。

## 開発環境

*   **フレームワーク**: Next.js (App Router)
*   **言語**: TypeScript
*   **スタイリング**: Tailwind CSS
*   **リンター**: ESLint (`npm run lint`)

## 注意事項

*   UIはモダンでスタイリッシュなダークモードを基調とします。
*   既存のコードの品質を維持し、一貫性のあるコーディングを心がけてください。
*   ユーザーからの指示が最優先です。このドキュメントと矛盾する場合は、ユーザーの指示に従ってください。
